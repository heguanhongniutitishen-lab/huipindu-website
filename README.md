# 慧拼读 AI英语单词学习系统官网

响应式 ToB 高转化官网，基于 Next.js、TypeScript、Tailwind CSS 和 Framer Motion。

## 运行

```bash
npm install
npm run dev
```

访问 `http://localhost:3000`。

## 结构

- `app/layout.tsx`: SEO metadata 与全局布局
- `app/page.tsx`: 首页入口
- `components/landing-page.tsx`: 首页模块与交互组件
- `app/globals.css`: 全局样式、玻璃拟态与布局工具类
- `tailwind.config.ts`: 品牌色、阴影、圆角配置
